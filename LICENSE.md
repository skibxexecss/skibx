## 🧩 Plugin Development Quick-Start

Skibx Executor 2026 exposes a lightweight plugin interface that lets you add custom commands, UI panels, and script hooks. Place your plugin file in the `./plugins` directory and Skibx Executor will load it at startup.

```lua
-- example_plugin.lua — minimal Skibx Executor plugin

local plugin = skibx.plugin.register({
    name = "HealthLogger",
    version = "1.0.0",
    author = "YourName"
})

plugin:on("inject", function(context)
    print("[HealthLogger] Injection fired in " .. context.game)
end)

plugin:on("tick", function()
    local hp = game.Players.LocalPlayer.Character
        and game.Players.LocalPlayer.Character:FindFirstChild("Humanoid")
        and game.Players.LocalPlayer.Character.Humanoid.Health
    if hp then
        skibx.log("HP: " .. tostring(hp))
    end
end)
```

> Place your file at `./plugins/HealthLogger.lua` and restart Skibx Executor — the plugin loads automatically.

## 🛠️ Troubleshooting Common Issues

- ❌ **Skibx Executor won't launch on Windows**
  The executable may be quarantined by your antivirus. Open Windows Security → Virus & Threat Protection → Protection History, restore the file, and add an exclusion for the install folder.
  **Solution:** Whitelist `SkibxExecutor.exe` in your antivirus and re-run as administrator.

- ❌ **Injection fails with "Timeout" error**
  This usually means Roblox loaded slower than the configured delay window.
  **Solution:** Increase `injection_delay` in `config.json` to `5000` or higher and relaunch Skibx Executor.

- ❌ **Skibx Executor APK crashes on launch (Android)**
  Older WebView versions can cause the UI to fail on first render.
  **Solution:** Update Android System WebView from the Play Store, then reopen the Skibx Executor APK.

- ❌ **Skibx Executor IPA shows "Untrusted Developer" (iOS)**
  This is a standard iOS sideloading prompt and not a security issue with Skibx Executor Roblox IOS.
  **Solution:** Navigate to **Settings → General → VPN & Device Management**, tap the developer profile, and select **Trust**.

- ❌ **Scripts execute but nothing happens in-game**
  The script may target a game API that has been updated since the script was written.
  **Solution:** Check the script hub for an updated version or review the Skibx Executor console for deprecation warnings.

- ❌ **High memory usage during long sessions**
  Running multiple heavy scripts without clearing state can accumulate memory.
  **Solution:** Use the **Clear Console** keybind (`Ctrl+L`) periodically and restart Skibx Executor every few hours during marathon sessions.
